LineageOS on OnePlus Nord N10

At this time there is no official image for the OnePlus Nord N10, so I'm piecing together bits of information from various sources.

Preparation (desktop)

  1. Install ADB tools on Ubuntu Lineage wiki
  2. Install android-sdk-platform-tools-common and check permissions (if udev permission errors appear later)
Run all `adb` commands from `~/adb-fastboot/platform-tools`
  1. Download OnePlus official build just in case

Preparation (phone)

  1. Charge and try out the new phone camera/wifi (not used sim card/ data)
  2. Upgrade to latest OxygenOS
  3. Unlock developer settings (seven taps on build number in system settings)
  4. Enable USB debugging (Developer settings)

Use the install wizard as if opening the box fresh from OnePlus.

Installing LineageOS unofficial image

  1. Download unofficial rom for OnePlus Nord N10/bille
  2. Download Magisk.apk
  3. Unlock the bootloader following instructions for the Nord N100
Enable USB debugging again after reboot.  Note that OEM Unlocking is now greyed out.
  1. Run adb devices to verify that we can talk to the device.
  2. Run adb reboot bootloader followed by fastboot devices to load the bootloader screen
  3. Run fastboot flash recovery <path/to/recovery.img> to send recovery to the device. 
Navigate to _Recovery Mode_ using the volume keys and press power button to apply
  1. From the Recovery mode screen select factory reset
  2. After restart and re-enable of USB debugging go back to bootloader screen
  3. Run fastboot boot <path/to/recovery.img>
  4. Navigate using the volume keys to Apply from ADB, at this point run adb sideload <path/to/rom.zip>
  5. Reboot device from the menu. Note that OnePlus startup logo has been replaced by LineageOS.
  6. Install Magisk following official instructions

I had various issues after trying this, ultimately realising that the forum post was written for a different Nord N10 model (BE2026 vs BE2029).
